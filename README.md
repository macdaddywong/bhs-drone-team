# bhs-drone-team
Coding examples for the Burlington High Drone Team

My class at Burlington High is learning OpenCV and Python for a drone competition.

This code was adapted from older code I had gotten from https://pyimagesearch.com

Unfortunately, that code no longer worked, so I used Microsoft Copilot to get it to work. Ultimately, we need to write some code that will recognize colors and then drop a life preserver that will be used to pick up objects for the drone competition.

I also looked at  https://medium.com/@carlosdnotes/install-opencv-and-jupyter-with-uv-76aa1e4581a3 
to set things up.

I'm also using uv.
https://docs.astral.sh/uv/
